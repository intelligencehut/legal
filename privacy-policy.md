# Privacy Policy

**Intelligence Hut Project Management Tool**
Last updated: February 18, 2026

## Overview

This Jira app ("Intelligence Hut Project Management Tool") is developed and maintained by Intelligence Hut Private Limited. This policy describes how the app handles user data.

## Data Collection and Usage

This app accesses the following Jira data solely to provide its functionality:

- **Jira Issues**: Issue keys, summaries, statuses, assignees, dates, and descriptions to display in the project spreadsheet view.
- **Worklogs**: Time tracking entries (time spent, dates, authors) to display and manage in the timesheet and issue panel views.
- **User Information**: Display names and avatars for user selection in filters and assignee fields.
- **Project Information**: Project names and keys for filtering and navigation.

## Data Storage

- This app does **not** store any Jira data externally.
- All data is accessed in real-time from Jira via the Atlassian Forge platform.
- The app uses Forge's built-in app storage only for app configuration preferences.
- No data is transmitted to third-party services.

## Data Sharing

We do **not** sell, share, or transfer any user data to third parties.

## Security

This app runs on the Atlassian Forge platform, which provides:
- Sandboxed execution environment
- Scoped API permissions
- User-level authorization via OAuth 2.0

## User Rights

Users can revoke the app's access at any time through their Atlassian account settings. Upon uninstallation, all app storage data is automatically deleted by the Forge platform.
